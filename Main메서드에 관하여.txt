public static void main(String[] args)

JAVA는 어플리케이션이 실행되면 제일먼저 메인(main)메소드를 실행한다.
public 접근제어자 (종류는 제약이 강한순서대로 private - protected - public)
                default는 클래스 내부와 동일 패키지에서 접근할 수 있다.
                public은 어느곳에서든 해당 객체를 참조할 수 있다는 의미

static은 이 함수가 정적 함수임을 의미한다.
static으로 함수 또는 클래스를 선언했을 경우에 해당 객체는 자바가 컴파일 되는 순간 정의가 된다.
static 이 먼저 정의가 되기때문에 아직 정의되지 않은 객체는 호출할 수 없다.
Static은 미리 메모리에 올려져있기 때문에 아직 메모리에 올라가지 않은
비 Static객체를 호출할 수 없다. main메소드에서 static이 아닌 다른 메소드를 호출하려고 해보면 오류가 난다.

void는 그냥 실행만 할뿐 나에게 다른걸 바라지마, 아무것도 돌려주지 않아
void는 리턴되는 값이 없다. 해당 함수가 종료된 후 호출한 부분으로 이동

이 메인메소드는 자바 어플리케이션 실행시 제일먼저 동자갛며 어느 객체에서든 접근할 수 있으며,
자바가 컴파일 되는 순간 정의되는 돌려주는 값이없는 함수.

String[] args
String은 문자열
Hello Wolrd와 같은 데이터의 형태를 String 이라는 클래스변수로 정의하겠다.

[]는 배열을 의미
연속적인 데이터를 하나의 변수에 넣고싶을때 즉 배열에 인덱스가 있고 [0][1][2]와 같이 인덱스로 구분되는 공간에 데이터를 넣겠다는 의미

args는 단순한 변수명 (아무 변수명이나 넣어도 상관이 없다.)
